# Синтетические эксперименты: первый основной результат

## 1. Постановка задачи

Рассматривается задача **data-driven анализа устойчивости и робастности** нелинейной динамической системы второго порядка.

Имеются траектории состояния `x(t) in R^2`, сгенерированные для синтетических эталонных нелинейных систем. По данным строится приближенная модель:

\[
\hat f(x) = \Theta(x) C,
\]

где `Theta(x)` — словарь базисных функций, `C` — матрица коэффициентов, оцененная методом наименьших квадратов.

Цели:

1. Оценить точность идентификации по residuals.
2. Оценить уровень неопределенности `epsilon` как 95%-квантиль нормы residual.
3. Выполнить локальный анализ устойчивости (Jacobian + Lyapunov).
4. Проверить робастное поведение модели с ограниченной аддитивной неопределенностью.
5. Добавить линейную feedback-связь и сравнить controlled/uncontrolled режимы.

---

## 2. Основные допущения

1. **Динамика гладкая** в рассматриваемой области состояний.
2. Доступны траектории без измерительного шума (синтетический стенд).
3. Полиномиальный словарь низкой степени достаточно хорошо аппроксимирует векторное поле в области наблюдений.
4. Неопределенность моделируется аддитивным возмущением
   \[
   \dot x = \hat f(x) + \Delta(t), \quad \|\Delta(t)\| \le \epsilon.
   \]
5. Для анализа устойчивости используется линеаризация в нуле:
   \[
   A = \left.\frac{\partial \hat f}{\partial x}\right|_{x=0}.
   \]
6. Для controlled-case используется статическая линейная обратная связь `u = -Kx`.

## 3. Метод (кратко)

### 3.1 Data generation

Для каждой системы моделируются несколько траекторий из разных начальных условий. Численно оцениваются производные `x_dot`.

### 3.2 Identification

Используется словарь:

\[
\Theta(x) = [1, x_1, x_2, x_1^2, x_1x_2, x_2^2, x_1^3, x_2^3].
\]

Коэффициенты `C` находятся из LS-задачи.

### 3.3 Residual analysis and epsilon

Residual:

\[
r = \dot x - \hat f(x).
\]

Далее считаются нормы `||r||`, RMSE и

\[
\epsilon = \text{quantile}_{0.95}(||r||).
\]

### 3.4 Lyapunov analysis

1. Строится `A`.
2. Проверяются собственные значения `A`.
3. Решается

\[
A^T P + PA = -Q, \quad Q=I.
\]

4. Рассматривается `V(x)=x^T P x` и её контуры в области `Omega` (bounding box по данным).

### 3.5 Robust simulation and control

Симулируется:

\[
\dot x = \hat f(x) + \Delta(t), \quad ||\Delta(t)||\le\epsilon.
\]

Затем controlled режим:

\[
\dot x = \hat f(x) + \Delta(t) - Kx.
\]

## 4. Экспериментальные системы

1. **Quadratic oscillator**
   \[
   \dot x_1 = x_2,\quad
   \dot x_2 = -\omega^2 x_1 - \alpha x_2 + \beta x_1^2.
   \]

2. **Van der Pol oscillator**
   \[
   \dot x_1 = x_2,\quad
   \dot x_2 = \mu (1-x_1^2)x_2 - x_1.
   \]

3. **Cross nonlinear system**
   \[
   \dot x_1 = -x_1 + x_2 + \gamma x_1x_2,\quad
   \dot x_2 = -2x_2 + \delta x_1^2.
   \]

4. **Saturation system**
   \[
   \dot x_1 = -x_1 + \tanh(x_2),\quad
   \dot x_2 = -x_2.
   \]
